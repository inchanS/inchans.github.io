개발환경

  • Javascript 런타임 플랫폼: Node.js
  • 언어: TypeScript
  • 프레임워크: Express
  • DB: MySQL
  • ORM: TypeORM  

현재 조건 상황

  • 회원가입시, 이메일과 패스워드가 필수 입력입력인데, 이메일의 경우 mySQL에서 unique 처리
