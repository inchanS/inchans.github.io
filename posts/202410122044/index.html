PopClip 앱이 맥 앱스토어에서 내려갔다.
이 사실을 늦게 알게된 바람에 약 1년간의 공백을 건너띄고 업데이트를 했더니 몇몇 Extension에서 문제들이 발생했다.
특히 가장 빈번하게 사용하던 번역 기능 확장들이 제대로 동작하지 않다보니 급한대로 만들었다.

PopClip Extension - deepL translation

PopClip Extensions Directory의 리스트에 있는 DeepL Translate Extension이 제대로 작동하지 않는다. (2024-10-10 기준)

해당 확장은 deepL의 App, Website 둘 중 하나를 선택하여 사용할 수 있도록 JS,TS로 동작하게 되어있는데 일단 Website 선택시 해당 페이지로 이동만 할 뿐, 선택된 텍스트가 전달되지 못하는 문제가 있다.
App으로 선택해보니 아예 앱이 열리지도 않았다.

현재 Extension이 제대로 동작하지 않는 현재 조건은 다음과 같다.

  • macOS 15.0.1
  • PopClip ver 2024.5.2

 

문제

  1. Website 사용의 경우 전달 URL의 해시값과 관련된 API가 변경된듯 보인다.
  2. APP 사용의 경우 앱 호출이 안된다.

 

PopClip Official Extension과의 차이점

우선 website로만 연결 가능

  1. 웹사이트를 통한 기능은 그대로 작동할 수 있도록 url 해시 수정
  2. APP 또는 Website를 고를 수 있는 불필요한 Option 제거(target Language 선택창은 그대로)
  3. 불필요한 APP 작동 코드 제거
  4. 대상언어의 기본값을 한국어로 변경
